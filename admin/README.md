# لوحة التحكم

هذه واجهة أولية فقط، بدون اتصال بقاعدة بيانات. الهدف منها تجهيز أماكن الإدارة قبل ربط المشروع لاحقًا مع Supabase.

## الجداول المقترحة

- `site_settings`: الهوية، الألوان، الشعار، وبيانات التواصل.
- `pages`: محتوى الصفحات العربية والإنجليزية.
- `services`: الخدمات وترتيبها وحالتها.
- `service_requests`: طلبات الخدمة القادمة من نموذج طلب الخدمة.
- `projects`: المشاريع والتصنيفات والصور.
- `blog_posts`: المقالات واللغة والرابط والكلمات الدلالية.
- `blog_categories`: تصنيفات المقالات.
- `contact_messages`: طلبات التواصل وحالة المتابعة.
- `media_assets`: الصور والملفات المرفوعة.

العربية تبقى النسخة الأساسية في جذر الموقع، والإنجليزية داخل `en/`.
